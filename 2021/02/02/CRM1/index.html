1.1 关于经典回归模型

如果我们的目的只是对数据进行直线拟合，那么机械地应用最小二乘法对 X 和 Y 进行拟合即可。 然而，我们的目的并不限于此，我们还需要知道估计值（a，b）和未知参数（α，β）之间的误差有多大。 扰动项的性质是什么？ 为了对这些问题得出一定的结论，我将通过总结整理、分析研究所学内容并在此基础上验证经济模型。 为了做到这一点，首先需要对 X 和 Y 的决定机制做出一定的假设。 我们即将做出的假设是用回归进行统计分析的起点。

首先，我们把 Y 和 X 的关系写成：Y=α+βX+ε

让我们假设这种关系满足以下 4-5 个假设条件
　假设 1：X 是非随机的
　假设 2：干扰项的期望值为零（E (ε)=0）
　假设 3：干扰项的方差是恒定的 (Var (ε) = σ² ) (同方差性)
　假设 4：干扰项之间互不相关（Cov (εi, εj)=0） (独立性)
　假设 5：ε 遵循正态分布 (正規性)

下面我来简单解释一下假设 1-5 的含义。

假设 1（X 是非随机的）
　该假设意味着解释变量（X）的值是由研究者预先设定的。例如，如果研究者向消费者提供预先确定的金钱（收入）并通过观察每个人的消费情况来收集数据，那么该假设是成立的。 虽然这种假设在实际经济数据中不太可能出现，但在这种假设下很容易得出估计函数的统计特性。有人可能会认为从这种不切实际的假设中得出的估计函数的特性在实践中是无用的，但即使在 X 是随机的情况下，根据假设 1 得出的结果几乎总是成立的。

假设 2（干扰项的预期值为 0）
　如果我们认为消费函数的干扰项代表了除收入以外影响消费的所有因素的总和，那么它的预期值可能会取一个非零值。
在这种情况下，E (ε)=c (≠0)
但如果我们改写 ε=c+ε’，
那么 E (ε’)=0，
模型就变成了：

  Y = α+βX+ε = α+βX+c+ε’ = α’+βX+ε’（其中α’= α+c）

该 ε’的期望值将始终为 0。
因此，假设干扰项的预期值为 0，根本不会改变模型的性质。

假设 3 同方差性（Homoskedasticity）
　这个假设意味着无论 X 的值是多少，Y 在 α+βX 附近的方差都是不变的。在消费函数的例子中，意味着收入为 200 万 rmb 和 2000 万 rmb 的两个家庭的消费方差是相同的。 当然，在现实中，消费的方差会随着收入的增加而增加，所以这个假设很可能不成立。

假设 4 独立性
　这一假设意味着，在截面数据的情况下，影响第 i 个家庭和第 j 个家庭消费的除收入以外的因素是相互独立的。例如，在下述情况下，该假设不成立：
　两户人家的孩子是朋友，两个家庭一起开 X’mas party
　在这种情况下，两个干扰项之间呈正相关。此外，在时间序列数据的情况下，假设某一年发生了对消费的负冲击（比如对未来收入的长期悲观预期）这种影响不仅会延续到当年的消费中，也会延续到未来的消费中。在这种情况下，干扰项之间也会呈正自相关。

假设 5 规范性
　这个假设意味着干扰项的分布遵循正态分布。假设不成立的情况，例如被解释变量只取整数值，如 X 是学习时间，Y 是考试分数。在这种情况下，干扰项是离散的，不是正态分布。 然而，在很多情况下，可以认为干扰项近似遵循正态分布。另外，在干扰项代表了收入以外的无数因素影响的总和的情况下（比如消费函数），根据中心极限定理，可以认为干扰项遵循正态分布。

假设 1-4 成立的模型称为经典（双变量）回归模型（CRM；Classical Regression Model）
如果在假设 1-4 的基础上加入假设 5，则该模型称为经典正态回归模型（CNRM; Classical Normal Regression Model）
